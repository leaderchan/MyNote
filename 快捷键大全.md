## Windows快捷键

### **一、Win相关快捷键**

- Win+R 运行
- Win+E资源管理器
- Win+T任务栏应用切换
- Win+D回到桌面
- Win+V打开粘贴板
- Win+L锁屏
- Win+方向键移动窗口位置
- Win+I打开设置
- Win+M最小化
- Win+Shift+M还原桌面所有窗口
- Win +Alt +F4关闭窗口
- Win+Tab切换窗口
- Win + (+/-)：打开Windows放大、缩小功能

### **二、Ctrl相关快捷键**

- Ctrl + Shift + N ：可快速创建新的文件夹
- Ctrl + Shift + Esc ： 快速打开 Windows 任务管理器。

- Ctrl+Home回到顶端
- Ctrl+End回到底端

- Ctrl+R刷新当前页面
- Ctrl+Home快速移到文件头部
- Ctrl+End快速移到文件尾部
- Ctrl+Esc显示开始菜单栏
- Ctrl+F5强行刷新
- Ctrl+ +（加号）放大页面
- Ctrl+ -（减号）缩小页面
- Ctrl+S保存
- Ctrl+T垂直平铺所有窗口
- Ctrl+V粘贴当前剪贴板内的内容
- Ctrl+W浏览器关闭当前窗口
- Ctrl+X剪切当前文本内容
- Ctrl+Y返回上一步
- Ctrl+Z撤消上一步
- Ctrl+Tab 小菜单方式向下切换标签

- Ctrl+K关闭除当前和锁定标签外的所有标签
- Ctrl+L打开“打开”面版
- Ctrl+N新建空白窗口
- Ctrl+P打开“打印”面板
- Ctrl+Q打开“添加到过滤列表”面板
- Ctrl+A全部选中当前页面内容
- Ctrl+C复制当前选中内容
- Ctrl+D打开“添加收藏”面版
- Ctrl+E打开或关闭“搜索”侧边栏
- Ctrl+F打开“查找”面版
- Ctrl+G打开或关闭“简易收集”面板
- Ctrl+B文本粗体
- Ctrl+H打开“历史”侧边栏
- Ctrl+Alt+Del打开任务管理器
- Ctrl+Alt+A QQ与QQ浏览器开启时截屏
- Ctrl+Alt+UP切换屏幕方向
- Ctrl+数字打开对应任务栏应用

### **三、Alt相关快捷键**

- Alt+Tab切换进程

- Alt+F4关闭当前程序
- Alt+Tab在桌面进行窗口切换
- Alt+Z切换自动换行
- Alt+Left浏览器里前进后退
- Alt+F打开文件菜单搜索
- Alt+V打开视图菜单
- Alt+O打开格式菜单
- Alt+A打开Excel表格菜单
- Alt+W打开窗口菜单

### **四、Shift相关快捷键**

- Shift+Home选中前面的文本
- Shift+End选中后面的文本
- Shift+Tab向前缩进

- Shift+Alt+S截取屏幕
- Shift+Alt+D新建窗口
- Shift+Alt+方向键切换窗口

- Shift+F3切换字母大小写 
- Shift+Spacebar（空格键）切换输入法 半全角 
- Shift+Ctrl+T还原上次关闭的标签页 
- Shift+Del永久删除文件 
- Shift+Ctrl+N新建文件夹 
- Shift+F10 选中文件的右菜单 
- Shift + Tab在选项之间往后移动

## IDEA快捷键

### 一、Ctrl相关快捷键

- Ctrl+N   查找类,可以快速打开类
- Ctrl+R 替换文本
- Ctrl+F 查找文本
- Ctrl+P 方法参数提示
- Ctrl+空格 代码提示
- Ctrl+X 删除行
- Ctrl+D 复制上一行到下一行
- Ctrl+J  自动代码
- Ctrl+E 最近打开的文件
- Ctrl+H 显示类结构图
- Ctrl+Q 显示注释文档
- Ctrl + O 快速重写方法
- Ctrl＋F12，可以显示当前文件的结构
- Ctrl+Up/Down 光标跳转到第一行或最后一行下
- Ctrl＋[或]可以跳到大括号的开头结尾
- Ctrl＋F7可以查询当前元素在当前文件中的引用，然后按F3可以选择
- Ctrl＋W可以选中单词继而语句继而行继而函数，连续按会有其他效果
- Ctrl+B 快速打开光标处的类或方法，选中单词参看帮助文档
- Ctrl + 光标位置：检验导入路径是否正确，正确变成手型
- Ctrl+E或者Alt+Shift+C 最近更改的代码

- Ctrl+Shift+N 查找文件，可以快速打开文件
- Ctrl+Alt+L  格式化代码
- Ctrl+Alt+O 优化导入的类和包
- Ctrl+Shift+Space 自动补全代码
- Ctrl+Alt+Space 类名或接口名提示
- Ctrl+Shift+Alt+N 查找类中的方法或变量
- Ctrl+Shift+向上键
- Ctrl+/ 或 Ctrl+Shift+/  注释（// 或者/*...*/ ）
- Ctrl+Alt+ left/right 返回至上次浏览的位置
- Ctrl+Shift+Up/Down 代码向上/下移动。
- Ctrl＋Shift＋Backspace可以跳转到上次编辑的地方
- Ctrl＋Shift＋Insert可以选择剪贴板内容并插入
- Ctrl + Shift + F10快速运行代码
- Ctrl + ALT +B快速进入接口实现类
- 选中文本，按Ctrl+Shift+F7 ，高亮显示所有该文本，按Esc高亮消失。
- Ctrl + Alt + V自动生成接收对象，可以引入变量。例如把括号内的SQL赋成一个变量
- Ctrl + ALT + T：快速使用trycatch等，把代码包在一块内，例如try/catch

### 二、Shift相关快捷键

- Shift+F6  重构-重命名
- Shift+F2或F2高亮错误或警告快速定位
- Shift＋Click可以关闭文件
- Shift + Enter光标快速移动到下一行

### 三、Alt相关快捷键

- Alt＋Q可以看到当前方法的声明
- Alt+1 快速打开或隐藏工程面板
- Alt + Enter自动导包

- Alt+回车 导入包,自动修正
- Alt+ left/right 切换代码视图
- Alt + Insert 可自动构造方法根据自己的需要进行选择
- Alt+ Up/Down 在方法间快速移动定位，在方法间快速移动
- Alt+Insert 生成代码(如get,set方法,构造函数等)
- 选中文本按Alt+F3 ，逐个往下查找相同文本，并高亮显示。
- Alt + 7打开一个窗口，能够看到类的所有信息
- Alt+Shift+C 对比最近修改的代码
- Alt+F1 查找代码所在位置，可以将正在编辑的元素在各个面板中定位

### 四、快速生成语句

- 快速生成迭代器：iter，回车或table
- 快速生成for循环：fori，回车或table

## Eclipse快捷键

- Shift + alt + s

- Alt + /

- ctrl+shift+o快速导包

- Ctrl+1 快速修复(最经典的快捷键)

- Ctrl+D 删除当前行

- Ctrl+Alt+↓ 复制当前行到下一行(复制增加)
- Ctrl+Alt+↑ 复制当前行到上一行(复制增加)
- Alt+↓ 当前行和下面一行交互位置(特别实用,可以省去先剪切,再粘贴了)
- Alt+↑ 当前行和上面一行交互位置(同上)
- Alt+← 前一个编辑的页面
- Alt+→ 下一个编辑的页面(当然是针对上面那条来说了)
- Alt+Enter 显示当前选择资源(工程,or 文件 or文件)的属性
- Shift+Enter 在当前行的下一行插入空行(这时鼠标可以在当前行的任一位置,不一定是最后)
- Shift+Ctrl+Enter 在当前行插入空行(原理同上条)
- Ctrl+Q 定位到最后编辑的地方
- Ctrl+L 定位在某行 (对于程序超过100的人就有福音了)
- Ctrl+M 最大化当前的Edit或View (再按则反之)
- Ctrl+/ 注释当前行,再按则取消注释
- Ctrl+O 快速显示 OutLine
- Ctrl+T 快速显示当前类的继承结构
- Ctrl+W 关闭当前Editer
- Ctrl+K 参照选中的Word快速定位到下一个
- Ctrl+E 快速显示当前Editer的下拉列表(如果当前页面没有显示的用黑体表示)
- Ctrl+/ (小键盘) 折叠当前类中的所有代码
- Ctrl+×(小键盘) 展开当前类中的所有代码
- Ctrl+Space 代码助手完成一些代码的插入(但一般和输入法有冲突,可以修改输入法的热键,也可以暂用Alt+/来代替)
- Ctrl+Shift+E 显示管理当前打开的所有的View的管理器(可以选择关闭,激活等操作)
- Ctrl+J 正向增量查找(按下Ctrl+J后,你所输入的每个字母编辑器都提供快速匹配定位到某个单词,如果没有,则在stutes line中显示没有找到了,查一个单词时,特别实用,这个功能Idea两年前就有了)
- Ctrl+Shift+J 反向增量查找(和上条相同,只不过是从后往前查)
- Ctrl+Shift+F4 关闭所有打开的Editer
- Ctrl+Shift+X 把当前选中的文本全部变味小写
- Ctrl+Shift+Y 把当前选中的文本全部变为小写
- Ctrl+Shift+F 格式化当前代码
- Ctrl+Shift+P 定位到对于的匹配符(譬如{}) (从前面定位后面时,光标要在匹配符里面,后面到前面,则反之)
- Alt+Shift+R 重命名 (是我自己最爱用的一个了,尤其是变量和类的Rename,比手工方法能节省很多劳动力)
- Alt+Shift+M 抽取方法 (这是重构里面最常用的方法之一了,尤其是对一大堆泥团代码有用)
- Alt+Shift+C 修改函数结构(比较实用,有N个函数调用了这个方法,修改一次搞定)
- Alt+Shift+L 抽取本地变量( 可以直接把一些魔法数字和字符串抽取成一个变量,尤其是多处调用的时候)
- Alt+Shift+F 把Class中的local变量变为field 变量 (比较实用的功能)
- Alt+Shift+I 合并变量(可能这样说有点不妥Inline)
- Alt+Shift+V 移动函数和变量(不怎么常用)
- Alt+Shift+Z 重构的后悔药(Undo)
